УСЛОВИЯ И ПОЛОЖЕНИЯ ТУРНИРА

Нажимая на кнопку “Участвовать” при регистрации, вы соглашаетесь со следующими Условиями пользования.

  1. Каждый трейдинговый турнир, который проводится на WhiteBIT, начинается в 00:00 UTC, и заканчивается в 23:59 UTC в дни, указанные для каждого турнира.
  2. Турнирная таблица лидеров обновляется один раз в день. Последнее обновление происходит в 00:00 UTC в последний день каждого турнира.
  3. Для каждого турнира, объем торгов пользователей будет считаться только по торговой паре (парам), указанным на страницах турнира.
  4. Объем торгов считается как общая сумма процентов PNL выполненных ордеров каждого пользователя на покупку и продажу на указанной торговой паре в период действия турнира.
